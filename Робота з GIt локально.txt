		Підготовка Git на ПК
1 Встановити Git з сайту 
2. Налаштувати конфігурацію командами (робиться один раз)
   git config --global user.name "name"
   git config --global user.email "email"
		Підготовка папки проекту
3. Зробити папку де будемо зберігати свій код
4. Відкрити цю папку в VS Code через Open Folder
5. Створити файли в папці
		Робота з Git
6. Відкрити в терміналі VS Code папку і там ввести git init
7. Після додавання коду зберігаємо файл а потім в терміналі набираємо
   git add name_file або git add .
8. Якщо не виникло помилок, то вводимо наступну команду
   git commit -m "Add other ..."
9. Для перевірки коміту вводимо команду git log
10. Для перевірки статусу використовуємо команду git status або git log --oneline
11. Для відкату до якогось коміту вводимо git reset --hard код коміту
   до якого хочемо відкатитися. Після цього закриваємо заклідку з файлом і
   потім його знову відкрити
